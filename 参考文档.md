# 高中立体几何交互练习平台 · 参考文档

> **版本**: v1.0  
> **创建日期**: 2026-05-30  
> **项目类型**: Web 3D 交互学习平台  
> **目标用户**: 高中生、数学教师、几何学习者

---

## 1. 项目概述

### 1.1 项目定位

基于 Three.js 的立体几何学习平台，提供沉浸式 3D 几何体交互体验，支持几何体查看、练习模式和做题记录三大核心功能。

### 1.2 核心价值

- **可视化学习**: 将抽象的立体几何概念转化为直观的 3D 模型
- **交互式探索**: 支持旋转、缩放、展开、截面等操作
- **练习系统**: 高考题型覆盖，难度分级，实时统计
- **学习追踪**: 自动记录练习历史，分析学习趋势

### 1.3 技术栈

| 技术 | 版本 | 用途 |
|------|------|------|
| Three.js | v0.160.0 | 3D 渲染引擎 |
| GSAP | v3.12.5 | 动画库 |
| Vite | v5.0.0 | 构建工具和开发服务器 |
| Material Design 3 | - | UI 设计系统 |
| gh-pages | - | GitHub Pages 部署 |

## 2. 功能需求

### 2.1 查看模式

#### 2.1.1 几何体支持

**必需几何体（10种）**:

| 键名 | 中文名 | 类型 | 顶点数 | 棱数 | 面数 |
|------|--------|------|--------|------|------|
| cube | 正方体 | 多面体 | 8 | 12 | 6 |
| rectangularBox | 长方体 | 多面体 | 8 | 12 | 6 |
| triangularPrism | 三棱柱 | 多面体 | 6 | 9 | 5 |
| tetrahedron | 四面体 | 多面体 | 4 | 6 | 4 |
| squarePyramid | 四棱锥 | 多面体 | 5 | 8 | 5 |
| hexagonalPrism | 六棱柱 | 多面体 | 12 | 18 | 8 |
| triangularPyramid | 三棱锥 | 多面体 | 4 | 6 | 4 |
| cylinder | 圆柱 | 曲面 | - | - | 3 |
| cone | 圆锥 | 曲面 | - | - | 2 |
| sphere | 球 | 曲面 | - | - | 1 |

**扩展几何体（可选）**:
- 正八面体
- 正十二面体
- 正二十面体
- 圆台

#### 2.1.2 3D 交互

| 操作 | 功能 | 实现方式 |
|------|------|----------|
| 左键拖拽 | 旋转几何体 | OrbitControls |
| 滚轮 | 缩放视图 | OrbitControls |
| 右键拖拽 | 平移视图 | OrbitControls |
| 双击 | 重置视图 | 自定义事件 |

#### 2.1.3 视图切换

- **正视图**: 从正面观察几何体
- **侧视图**: 从侧面观察几何体
- **俯视图**: 从上方观察几何体
- **透视图**: 默认 3D 透视视角

#### 2.1.4 显示模式

| 模式 | 说明 | 实现 |
|------|------|------|
| 实体模式 | 标准渲染 | MeshStandardMaterial |
| 线框模式 | 仅显示棱 | WireframeGeometry |
| 虚线模式 | 虚线显示棱 | DashedLineMaterial |
| 点模式 | 仅显示顶点 | PointsMaterial |

#### 2.1.5 辅助元素

- **地面网格**: 可切换显示/隐藏
- **坐标轴**: 可切换显示/隐藏
- **顶点标记**: 可选显示顶点标签
- **棱标记**: 可选显示棱的标签

#### 2.1.6 透明度调节

- 滑块控制几何体透明度（0% - 100%）
- 支持单面透明度调节

#### 2.1.7 展开/折叠动画

**核心需求**:
- 支持多面体的展开图展示
- 动画过程流畅（使用 GSAP）
- 展开后可折叠回原状
- 展开过程中保持面的连接关系

**实现要点**:
- 使用 pivot/hinge 数据定义旋转轴
- 按 parent 关系构建折叠树
- 每个面围绕 hinge 轴旋转正确角度
- 动画进度 0-1 控制展开程度

#### 2.1.8 截面工具

**功能需求**:
- 选择 3 个顶点定义截面
- 实时预览截面位置
- 生成截面几何体
- 显示截面面积

**交互流程**:
1. 点击"截面工具"按钮进入模式
2. 在 3D 场景中点击顶点选择
3. 选择 3 个顶点后自动生成截面
4. 可清除重新选择

### 2.2 练习模式

#### 2.2.1 题型支持

| 题型 | 说明 | 输入方式 |
|------|------|----------|
| 填空题 | 填写数值或表达式 | 文本输入框 |
| 选择题 | 单选或多选 | 选项按钮 |
| 证明题 | 书写证明过程 | 文本编辑器 + 数学符号工具栏 |
| 计算题 | 计算结果 | 文本输入框 |

#### 2.2.2 难度分级

| 难度 | 说明 | 题目特征 |
|------|------|----------|
| 简单 | 基础概念 | 直接应用公式 |
| 中等 | 综合应用 | 需要多步推理 |
| 困难 | 高考难度 | 复杂证明或计算 |

#### 2.2.3 练习模式

- **当前几何体**: 仅出与当前显示几何体相关的题目
- **随机混合**: 从所有几何体中随机出题

#### 2.2.4 题目数量

- 可选：3、5、8、10 题
- 支持自定义数量

#### 2.2.5 实时统计

- 总题数
- 已完成题数
- 正确数
- 正确率（百分比）
- 用时

#### 2.2.6 提示系统

**多级提示**:
- **Level 1（💡概念）**: 回忆相关公式/定理
- **Level 2（🔍思路）**: 解题方向
- **Level 3（📝格式）**: 证明题的书写格式

#### 2.2.7 证明题工具栏

- 数学符号按钮（28个常用符号）
- 符号分类：关系符、运算符、希腊字母等
- 点击插入到文本框

#### 2.2.8 3D 解析可视化

- 答题后展示 3D 动态演示
- 动画效果（GSAP）
- 解析步骤分步展示

---

### 2.3 做题记录

#### 2.3.1 自动记录

- 记录每次练习的详细信息
- 包括：题目、答案、用时、正确性

#### 2.3.2 统计信息

- 总练习次数
- 平均正确率
- 总用时
- 各题型正确率
- 各难度正确率

#### 2.3.3 趋势图表

- 正确率趋势折线图（Canvas 绘制）
- 按时间维度展示学习进步

#### 2.3.4 导出/导入

- 导出 JSON 格式记录
- 导入 JSON 恢复记录
- 支持数据备份和迁移

## 4. 非功能需求

### 4.1 性能要求

- 首次加载时间 < 3 秒
- 3D 渲染帧率 ≥ 30fps
- 动画流畅度 ≥ 60fps
- 内存占用 < 200MB

### 4.2 兼容性

- **浏览器**: Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
- **WebGL**: 支持 WebGL 2.0
- **设备**: 桌面、平板、手机

### 4.3 可访问性

- 键盘导航支持
- 屏幕阅读器支持
- 高对比度模式

### 4.4 数据安全

- 本地存储练习记录
- 支持数据导出备份
- 无用户隐私数据收集
