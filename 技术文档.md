## 2. 核心模块实现

### 2.1 SceneManager（场景管理器）

**职责**: 管理 Three.js 场景、网格、坐标轴

**接口**:
```javascript
class SceneManager {
  constructor(container)
  
  // 场景管理
  getScene(): THREE.Scene
  getContainer(): HTMLElement
  
  // 辅助元素
  toggleGrid(): void
  toggleAxes(): void
  get gridVisible(): boolean
  get axesVisible(): boolean
  
  // 清理
  dispose(): void
}
```

**实现要点**:
- 使用 `THREE.GridHelper` 创建地面网格
- 使用 `THREE.AxesHelper` 创建坐标轴
- 坐标轴包含线条、箭头、标签，打包到 `axesGroup`
- 状态属性 `gridVisible` 和 `axesVisible` 用于 UI 同步

### 2.4 GeometryFactory（几何体工厂）

**职责**: 根据配置创建 Three.js 几何体

**接口**:
```javascript
class GeometryFactory {
  constructor()
  
  // 创建几何体
  createGeometry(config: GeometryConfig): THREE.Group
  
  // 几何体操作
  setDisplayMode(mode: 'solid' | 'wireframe' | 'dashed' | 'points'): void
  setOpacity(opacity: number): void
  
  // 清理
  disposeGroup(group: THREE.Group): void
}
```

**几何体配置**:
```javascript
interface GeometryConfig {
  key: string           // 键名，如 'cube'
  name: string          // 中文名，如 '正方体'
  type: 'polyhedron' | 'curved'  // 类型
  vertices: number      // 顶点数
  edges: number         // 棱数
  faces: number         // 面数
  createFunction: () => THREE.BufferGeometry  // 创建函数
}
```

### 2.5 CrossSectionManager（截面工具管理器）

**职责**: 管理截面选择和生成

**接口**:
```javascript
class CrossSectionManager {
  constructor(scene, geometryFactory)
  
  // 模式控制
  enterMode(): void
  exitMode(): void
  isActive(): boolean
  
  // 顶点选择
  handleClick(event: MouseEvent): void
  updatePreviewPoint(event: MouseEvent): void
  toggleVertexSelection(vertexName: string): void

  createSection(): void
  clearSection(): void
  
  // 回调
  onModeChange: (active: boolean) => void
  onPointsChange: (points: THREE.Vector3[]) => void
  onVertexSelect: (vertexName: string, selected: boolean) => void
}
```

**实现要点**:
- 使用 Raycaster 检测鼠标点击
- 支持顶点吸附（snap to vertex）
- 实时预览截面位置
- 使用 Earcut 库进行三角剖分

### 2.6 UnfoldManager（展开/折叠管理器）

**职责**: 管理几何体展开动画

**接口**:
```javascript
class UnfoldManager {
  constructor(scene, geometryFactory)
  
  // 展开控制
  unfold(): void
  fold(): void
  toggle(): void
  
  // 动画
  animate(progress: number): void  // 0 = 折叠, 1 = 展开
  isUnfolded(): boolean
  
  // 清理
  dispose(): void
}
```

**展开算法**:
1. 从 `unfoldConfig` 读取 pivot/hinge 数据
2. 按 parent 关系构建折叠树
3. 每个面围绕 hinge 轴旋转
4. 使用 GSAP 动画驱动 progress 从 0 到 1
5. `updateDisplay(progress)` 应用旋转

## 6. 性能优化

### 6.1 渲染优化

- 使用 `BufferGeometry` 代替旧版 `Geometry`
- 合并静态几何体减少 draw calls
- 启用 frustum culling
- 使用 LOD（Level of Detail）处理复杂场景

### 6.2 内存管理

- 及时 dispose 不使用的 geometry 和 material
- 使用 `disposeGroup()` 递归清理
- 避免内存泄漏

### 6.3 加载优化

- 代码分割（Code Splitting）
- 懒加载非关键模块
- 使用 CDN 加载第三方库
